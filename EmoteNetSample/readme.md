#NekoSprite
##Project AZUSA

这是一个E-mote.NET配合SharpDX.Direct3D9的示例。

E-mote.NET正在开发中，当前暂无更多说明。

为方便调试，提供一个PSB立绘文件。[下载](http://pan.baidu.com/s/1i3ioIiD "下载")

## 编译提示 ##

需要导入SharpDX的相关库，包括`SharpDX.dll` `SharpDX.Direct3D9.dll`。可以从Nuget取得。
